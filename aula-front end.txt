componente no react é uma função que retorna HTML

quando o HTML está integrado dentro do javascript ele se chama JSX

propriedades no react é igual a atributos no HTML, só que aplicados aos componentes

Estado é uma informação que vai ser mantida pelo componente

Imutabilidade é o fato de não poder se alterar uma variável. Para que uma informação
possa ser alterada dentro do componente é necessário usar um estado, que por definição re-
torna um array com seu valor e uma função de atualização que fica responsável por mudar o
valor do estado.
